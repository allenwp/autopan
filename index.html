Auto Pan for Discord

An auto-pan for Discord on Windows. Performs an automatic stereo spread of user's voices in a Discord voice channel.

Version 0.1 Pre-Alpha (No audio output yet!)

This release is a preview of the GUI functionality. It can connect and disconnect from a discord voice channel and has all logic in place to allow control of user's volumes and pan, including the auto pan feature. Except for audio output, this version is very close to what 1.0 will look like and how it will behave.

Status

Apr. 15, 2016 - Pre-alpha release: Has all functionality except for audio output. I'm waiting on Discord.Net for complete audio support before adding this (obviously crucial) feature.
